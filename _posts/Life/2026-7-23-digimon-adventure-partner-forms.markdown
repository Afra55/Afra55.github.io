1999 年的 **《数码宝贝大冒险》（Digimon Adventure）**，就是大家常说的「第一部」。八个孩子被卷入数码世界，各自与一只伙伴数码兽相遇；伙伴会随羁绊与危机，一级级**进化**成更强形态。

本文只讲第一部世界观里这八条主线：每个形态写清**中文常用名 / 日文 / 英文**，并附进化示意插图。想查设定细节，可对照 [Wikimon](https://wikimon.net/Digimon_Adventure) 与 [DigimonWiki – List of Partner Digimon](https://digimon.fandom.com/wiki/List_of_Partner_Digimon)。

## 先看这 5 条（没看过也能往下翻）

1. **第一部 = Digimon Adventure（1999）**，主角是**八神太一**，伙伴是**亚古兽**。
2. **进化等级**常见写法：幼年期 II → 成长期 → 成熟期 → 完全体 → 究极体（英文常称 In-Training / Rookie / Champion / Ultimate / Mega）。
3. **完全体**多靠徽章（纹章）觉醒；太一误用勇气会引出黑暗进化**骷髅暴龙兽**。
4. **究极体**里，太一与大和的**战斗暴龙兽 / 金属加鲁鲁兽**及合体**奥米加兽**在第一部本篇与剧场版最完整；其余几条的究极体多在后续同世界观作品（如 *tri.*）才正式亮相，本文仍按「Adventure 连续体」一并列出，方便对照。
5. **中英译名不统一**：如巴达兽 / Biyomon、迪路兽 / Gatomon（日文 Tailmon）。下文以大陆常见译名为主，并附英文。

## 一、八神太一 × 亚古兽（勇气）

太一的伙伴线是第一部辨识度最高的一条：橙恐龙 → 盔龙头暴龙 → 机械暴龙 → 装甲战士。中间还夹着一次「勇气用错」的黑暗进化。

![亚古兽进化线示意：球球兽→亚古兽→暴龙兽→机械暴龙兽→战斗暴龙兽]({{ site.baseurl }}/blog_picture/digimon-adventure-forms.resources/line-agumon.jpg)

### 特殊形态：黑暗进化与奥米加兽

![左侧骷髅暴龙兽、右侧奥米加兽示意]({{ site.baseurl }}/blog_picture/digimon-adventure-forms.resources/special-skull-omni.jpg)

*左：骷髅暴龙兽（黑暗进化）· 右：奥米加兽（与金属加鲁鲁兽合体）*

- **骷髅暴龙兽**：太一急于变强、强行催进化时出现——不是「更强的正轨」，而是失控的黑暗形态。
- **奥米加兽（Omnimon / Omegamon）**：战斗暴龙兽 + 金属加鲁鲁兽合体，剧场版《我们的战争游戏！》对抗迪亚波罗兽时登场，是第一部最经典的合体形态之一。

## 第一部本篇 vs 后续作品（读形态时别混）

| 形态 | 第一部本篇（约 54 集）观感 | 补充说明 |
| --- | --- | --- |
| 各成熟期 | 几乎全员都有完整戏份 | 开岛篇密集登场 |
| 太一 / 大和完全体与究极体 | 主线核心战力 | 含骷髅暴龙兽、奥米加兽（剧场版） |
| 阿武神圣天使兽、光天女兽 | 本篇关键战登场 | 天使系名场面 |
| 空 / 光子郎 / 美美 / 丈的究极体 | 本篇未全部铺完 | 多在 *Adventure 02* 能量补完、*tri.* 等补齐 |

所以：若你只看 1999 电视本篇，记得「形态图鉴」比「每一集都出现过」更宽；本文按 Adventure 连续体的伙伴线整理，方便一次查全。
